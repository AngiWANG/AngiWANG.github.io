| Ctrl+Shift+L   | 格式化             |
| Ctrl+Q         | 查看注释            |
| F8             | step over       |
| F7             | step into       |

| shift+F8       | step out        |
| F9             | resume program  |
| alt+F9         | run to cursor   |

| alt+enter      | import          |
