## 组（group）

```shell
# 添加ehts组
$ groupadd ehts
```

## 用户（user）

```shell
$ useradd -d /home/eht -g ehts eht
```

重置用户密码

```shell
# passwd eht
```

### SFTP用户

```shell
$ useradd -s /sbin/nologin -d /home/testsftpuser -g sftpusers testsftpuser
```
