## 执行sql文件
方法一
```shell
$ mysql -h localhost -u root -p123456 < F:\test\test.sql
```
*备注：执行后没有结果提示信息。*

方法二
```shell
# 先登录到mysql
mysql>source F:\test\test.sql
```

方法三
```shell
# 先登录到mysql
mysql>\. F:\test\test.sql
```
