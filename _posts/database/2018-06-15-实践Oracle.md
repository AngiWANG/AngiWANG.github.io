启动oracle数据库



```shell
# oracle安装用户下，一般是oracle
# 启动监听器
$ lsnrctl start
```



```shell
# 以dba身份登录并连接后
SQL> startup
SQL> quit
```


停止oracle数据库

```shell
# oracle安装用户下，一般是oracle
# 停止监听器
$ lsnrctl stop
```



```shell
# 以dba身份登录并连接后
SQL> shutdown
或
SQL> shutdown immediate
SQL> quit
```
