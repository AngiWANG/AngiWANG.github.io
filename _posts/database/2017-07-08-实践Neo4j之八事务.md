## Transaction

```java
try (Transaction tx = graphDb.beginTx()) {
	user1 = this.graphDb.createNode();
    logger.info("created user:" + user1.getId());
    tx.success();
}
```

or

```java
Transaction tx = graphDb.beginTx();
    try {
        user1 = this.graphDb.createNode();
        logger.info("created user:" + user1.getId());
      	//成功
        tx.success();
      	//如果失败则
      	//tx.failure();
    } finally {
      	tx.finish();
}
```
