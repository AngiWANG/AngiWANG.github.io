### 安装说明

1. fork库到自己的github
2. 根据自己的用户名（username）修改项目名字为：`username.github.io`
3. clone库到本地
4. 修改CNAME或者删掉这个文件，使用默认域名
5. 修改`_config.yml`配置项
6. It's done!
